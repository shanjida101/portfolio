# Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Interactive neon cyberpunk UI
- Responsive layout for mobile and desktop
- Contact card and project sections

## Usage

1. Open `index.html` in a browser.
2. Modify content in `index.html`, `style.css`, and `script.js` as needed.
